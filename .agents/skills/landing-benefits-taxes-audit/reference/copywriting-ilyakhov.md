## 1. Чувственный опыт

**Принцип:** убедительный текст опирается на чувственный опыт читателя — то, что можно увидеть, услышать, почувствовать. Читатель «смотрит внутренний фильм»; абстракции фильм не запускают.

**Чувственный опыт** — конкретная сцена: «ногам холодно», «менеджер послал клиента», «Ом напомнит о задаче в срок».

**Абстракция в тексте** — 🔴 налог: «управление ресурсами», «комплексное решение», «повышение эффективности». Не представить, не поверить.

### Как переводить абстракцию в выгоду

| Абстракция (🔴) | Чувственный опыт (🟢) |
|-----------------|----------------------|
| Управлять совместной работой | Давать задания и проверять, кто как работает |
| Повышать качество сервиса | Сразу отвечать на звонки и соединять с нужным менеджером |
| Эффективные решения бизнес-задач | Сделаем уборку в офисе за ночь: ковры, окна, пыль |
| Любую вещь купить и продать | Старые айфоны, велосипеды, мебель из Икеи, антиквариат |

### Сценарии вместо охвата

Не «решаем любые маркетинговые задачи» → опиши 2–3 узнаваемых сценария аудитории:

- «Низкая конверсия — вам ко мне»
- «Не знаете, как выйти на рынок — вам ко мне»

Конкретный список не отсекает остальное — читатель догадается. Формализм пугает сильнее, чем неполнота.

### Когда текст не нужен

Если сцену проще показать — скринкаст, видео, скрин интерфейса сильнее абзаца. Отсутствие визуала при сложном продукте — 🔴 налог.

### Допустимая абстракция

Абстракция + чувственное пояснение рядом — ок:

> «Контроль качества сервиса: менеджеры вовремя отвечают на письма и вежливы»

Без пояснения — 🔴 налог.

## 2. Заголовки

### Типы заголовков

| Тип | Вопрос | На лендинге | Пример |
|-----|--------|-------------|--------|
| **Назывной** | О чём? | Навигация, второстепенные блоки | «Тарифы», «Как это работает» |
| **Транзитивный** | Что сказано? | Hero, блоки-аргументы, CTA-секции | «Выручка выросла на 13%» вместо «Результаты 2014 года» |
| **Ни о чём** | — | 🔴 налог всегда | Кликбейт без связи с содержанием |
| **Игровой** | — | Редко, осознанно | «Утюг в подарок» для статьи про акции банков |

**Для лендинга:** hero и ключевые блоки — **транзитивные**. Читатель понимает главное, не читая абзац. Если к фрагменту нельзя написать транзитивный заголовок — в блоке несколько равнозначных идей (🔴).

Примеры транзитивности:

- «Динамика прибыли в 2011 г.» → «Рост прибыли в 2011 г. — 11%»
- «О компании» → «Дизайн продуктов, сайтов и интерфейсов с 2004 года» (если это правда и важно)

Транзитивный заголовок может быть длиннее назывного — всё равно короче тела блока.

### Пирамидальная структура сайта

На лендинге каждый блок = **вывод заголовком** + подробности ниже:

```
[Транзитивный заголовок = главная мысль блока]
Подробности. Подробности. Подробности.
[CTA]
```

Не «О нас» + абзац про миссию. А «Помогаем сдавать работу в срок — 12 000 команд» + как именно.

### Согласованность подзаголовков

Одноранговые подзаголовки — **одного типа** (все назывные или все транзитивные). Синтаксическая параллельность: «Не срывать сроки» / «Грамотно отвечать на звонки» / «Находить бездельников».

Избегай нарочитой параллельности, когда подзаголовок имеет смысл только вместе с общим заголовком (работает на плакате, не на длинной странице).

### Контраст заголовка

Заголовок визуально контрастен телу — иначе мозг его пропустит (🔴 налог внимания).

## 3. Оценочные суждения и усилители

**Оценочное суждение** — субъективная оценка автора: прилагательные и наречия без опоры на факт. Каждый поймёт по-своему → не доверяют → 🔴 налог.

**Самая прочная оценка** — та, которую читатель делает сам на основе фактов.

### Маркеры оценочности (🔴)

**Прилагательные:** качественный, надёжный, удобный, эффективный, инновационный, лучший, уникальный, профессиональный.

**Наречия:** быстро, удобно, максимально, очень, абсолютно, безусловно, весьма.

**Усилители** — оценка на оценке (хуже обычной оценки): «максимально выгодные условия», «очень качественное обслуживание», «настоящие профессионалы», «невероятная точность». Сигнал: автор очень старается убедить → 🔴 тяжесть 2–3.

**Слова-маркеры фальши:** действительно, вправду, по-настоящему, гарантированный, шокирующий, невероятный, уникальный, лучший.

### Что ставить вместо оценки (🟢)

1. **Факты и цифры:** «Безлимит за 350 ₽/мес» вместо «максимально выгодные тарифы»
2. **Сценарий / история:** как в примере с шоколадом — процесс, сроки, происхождение
3. **Доказательства:** отзыв с именем, скрин, ссылка, статистика

| Было (🔴) | Стало (🟢) |
|-----------|------------|
| Быстрый заработок | 10 000 ₽ за 40 часов работы |
| Качественный контент | Тексты проверяет редактор с 8-летним опытом в «Ведомостях» |
| У нас максимально выгодные тарифы | Безлимитный интернет, фильмы и игры — 350 ₽/мес |
| Настоящие профессионалы | [имя, кейс, цифра результата] |

### Оценка своей работы

«Мы — лидеры», «лучший сервис» — 🔴. Вместо этого: результаты, ссылки, примеры, статистика.

## 4. Связь с выгодами и налогами

| Приём Ильяхова | В методе выгод/налогов |
|----------------|------------------------|
| Чувственный опыт, сценарий, цифры | 🟢 выгода: читатель видит результат |
| Абстракция, штамп, канцелярит | 🔴 налог: когнитивная нагрузка |
| Оценочное суждение без факта | 🔴 налог: нет доверия |
| Усилитель («очень», «максимально») | 🔴 налог тяжесть +1: фальшь |
| Транзитивный заголовок hero | 🟢 выгода сила 3: суть с первого экрана |
| Назывной заголовок «О компании» в hero | 🔴 налог: не отвечает «что получу» |
| Заголовок «ни о чём» | 🔴 налог тяжесть 3 |
| Скринкаст / скрин вместо абзаца | 🟢 выгода: чувственный опыт напрямую |
| Формализм («любые задачи», «все услуги») | 🔴 налог: не представить |

## 5. Чеклист текстового аудита

Для каждого текстового элемента лендинга проверь:

- [ ] **Чувственный опыт** — можно ли представить сцену использования?
- [ ] **Сценарий** — описана ли ситуация аудитории, а не «все случаи жизни»?
- [ ] **Заголовок транзитивный** — понятно «что сказано» без чтения тела?
- [ ] **Нет оценок без фактов** — убраны прилагательные/наречия или подкреплены цифрой?
- [ ] **Нет усилителей** — «очень», «максимально», «настоящий»?
- [ ] **Нет штампов** — «инновационный», «комплексный», «индивидуальный подход»?
- [ ] **Визуал** — там, где текст слаб, есть скрин/видео?
- [ ] **Переписывание** — в рекомендации дать «было → стало» по правилам выше

## 6. Шаблон переписывания в рекомендациях

```
Проблема: {абстракция / оценка / назывной заголовок}
Тип: {чувственный опыт | заголовок | оценочное суждение}
Налог: 🔴 тяжесть {N}

Было: «{цитата с страницы}»
Стало: «{вариант с чувственным опытом / транзитивным заголовком / фактом}»
Почему лучше: {одна фраза}
```
