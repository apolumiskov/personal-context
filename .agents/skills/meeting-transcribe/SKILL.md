# Стенограмма встречи

Создание стенограммы онлайн встречи на основе видео или аудио файла и составление сотавление summary этой встречи.

## Когда использовать

- Когда тебя просят подготовить summary встречи.

## Параметры для работы

- Для работы ты должен получить файл записи или путь к этому файлу, если тебе его не передали ты не можешь продолжать работу.

## Инструкции

1. Проанализируй полученный файл для транскрибации. Если файл в видео-формате, первым делом тебе нужно извлечь и сохранить аудио-дорожку из этого файла. Для извлечения аудио-дорожки используй скрипт `scripts/audio-extract/run.py`. Для запуска python используй `python3`. Результат извлечения дорожки будет сохранен в директории `tmp/`, путь к файлу будет выведен скриптом

2. Если переданный файл является аудио файлом или если извлечение аудио-дорожки прошло усепшно - транскрибируй с помощью скрипта `scripts/transcribe/run.py`. Результат извлечения дорожки будет сохранен в директории `tmp/`, путь к файлу будет выведен скриптом. Если скрипт вернул сообщение `Switch on VPN` и код 1 - это значит нужно включить VPN. Останови работу и сообщи оператору, что нужно включить VPN.

3. После успешного создания транскрибации, составь краткое описание встречи на основе этой транскрибации (путь к файлу с транскрибацией вернет скрипт) используй собственную ИИ-модель. Примерный шаблон Summary `assets/meeting.md`. Проанализируй шаблон и сохрани в директории в которой лежит оригинальный файл. Формат имени для сохранения как оригинальное имя файла, тольок в md формате.

4. После успешного создания транскрибации удали все временные файлы и директории в директории навыка `tmp/`
