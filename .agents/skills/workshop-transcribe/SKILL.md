# Коспектирование учебных материалов

Создание конспекта учебных материалов на основе аудио-файла.

## Когда использовать

- Когда тебя просят сделать конспект урока.

## Параметры для работы

- Для работы ты должен получить файл записи (аудио) или путь к этому файлу, если тебе его не передали ты не можешь продолжать работу.
- Вспомогательный конспект лекции, это не обязательный параметр, а вспомогательный, используй его как дополнительный матерал, который будет взят за основу для создаваемого конспекта.
- Ссылка на оригинальную лекцию. Если ты получил этот параметр - укажи его в конспекте. Заголовок ссылки - "Оригинальное видео" 

## Инструкции

1. Переданный файл является аудио файлом, тебе необходимо извлечь из него аудиодорожку и транскрибировать с помощью скрипта `scripts/transcribe/run.py`. Результат извлечения дорожки будет сохранен в директории `tmp/`, путь к файлу будет выведен скриптом. Если скрипт вернул сообщение `Switch on VPN` и код 1 - это значит нужно включить VPN. Останови работу и сообщи оператору, что нужно включить VPN.
2. Составь конспект учебной лекции на основе транскрибации и других переданных материалов по уроку. При составлении конспекта учитывай тот факт, что лекция может содержать слова-паразиты, отвлчения и оффтопик. Такого рода мусор лучше убирать. Если в лекции приводятся примеры приводи их.
3. Сохрани конспект по тому же пути, где лежит аудио-файл
4. После создания конспект удали все содержимое директории `tmp`

### Рекомендации
- При создании конспекта не нужно ссылаться на файлы записей или вспомогательных конспектов.
