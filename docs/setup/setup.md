# Первичная инициализация окружения

Данная инструкция описывает первичную инициализацию персонального окружения. Однажды настроенное окружение не требует настройки при подключении к новым проектам в рамках одной машины.

## Базовое описание

Директория `.agents` является канонической директорией с настройками

## Инициализация

Перед началом работы нужно выполнить следующие действия:

- Обеспечить совместимость с агентской средой, путем создания символической ссылки на рабочую директорию агента;
- Добавить рабочую директорию агента в локальное исключение из git (**не .gitignore**);

```shell
ln -sfn .agents <agent_workdir>
```

```shell
echo "<agent_workdir>" >> .git/info/exclude
```

если <agent_workdir> уже отслеживается в git, выполнить:


```shell
git rm --cached -r <agent_workdir>
```

**Таблица имен рабочих директорий агентов для <agent_workdir>**
| Агентская среда | Рабочая директория |
|-----------|-----------|
| Cursor | .cursor |
